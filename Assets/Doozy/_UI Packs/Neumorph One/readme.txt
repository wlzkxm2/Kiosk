GETTING STARTED
---
After import, open the UIMenu window 
Tools > Doozy > UIMenu
and click the bottom left button 'Regenerate the UI Menu'.

This operation will do a search in your project for all UIMenuItems to add them to the internal UIMenu database.

Note that you can always run a Global Refresh
Tools > Doozy > Global Refresh
